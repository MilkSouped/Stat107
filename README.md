Project Overview
================
- **Scientific Question**: What differentiates top-rated products form bottom-rated products across various categories, and what keywords in customer reviews signal positive sentiments?
- **Group Members**: Ethan Chan, Alex Back, Darren Kuboyama
- **Summary of Project**: This project aims to analyze the Sephora Products and Skincare Reviews dataset by creating bar graphs and scatterplots to represent graphical trends for the top 3 and worst 3 products based on various categories.
- **Required Packages**:
    - tidyverse
        - dplyr 
    - ggplot2
